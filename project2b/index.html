Conceptual Design

The conceptual design will provide an idea of the abstract of the website. It lists the main features of the website.

Website Plan

The website plan will give an idea of the different media types like graphics, video and simple html files that will be used in the website. It illustrates which particular page will have what type of media type and whether special html forms will be used.

Hyperlink Structure

The website structure is similar to the conceptual design, but it further elaborates on how many sections apart from home page will be involved and the exact number of sub-section pages. It points out to the different layers of the website more deeply. It identifies the hyperlink structure of the website.
